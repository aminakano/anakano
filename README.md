# Ami Nakano

### Tech

### Installation and setup

Install the dependencies and devDependencies and start the server.

```sh
$ npm install
```

#### Compiles and hot-reloads for development

```sh
$ npm run dev
```
```sh
http://localhost:3000
```

#### Compiles and minifies for production

```sh
$ npm run export
```
